# Python Minesweeper

A terminal-based Minesweeper game developed in Python 
as part of coursework at the University of the 
Philippines Diliman.

## Features
- Random mine generation on a 7x7 grid
- Cell opening with adjacent mine counting
- Flag and unflag cells
- Defuser power-up that neutralizes mines
- Win and loss detection
- System displays mines remaining in real time

## How to Play
1. Run the game: python minesweeper.py
2. Choose an action:
   - **[O]** Open a cell
   - **[F]** Flag/Unflag a cell
   - **[D]** Use Defuser (when acquired)
3. Enter coordinates in format **[A-G][1-7]** (e.g. `B3`)
4. Avoid mines and reveal all safe cells to win!

## Technologies Used
- **Language:** Python 3
- **Concepts:** 2D arrays, recursion, game state management,
  randomization, input validation
